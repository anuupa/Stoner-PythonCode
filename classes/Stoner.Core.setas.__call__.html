setas.__call__¶

setas.__call__(*args, **kargs)[source]¶

Treat the current instance as a callable object and assign columns accordingly.

Variois forms of this method are accepted:

setas("xyzuvw")
setas(["x"],["y"],["z"],["u"],["v"],["w"])
setas(x="column_1",y=3,column4="z")
