DefaultPlotStyle.customise_axes¶

DefaultPlotStyle.customise_axes(ax, plot)[source]¶

This method is run when we have an axis to manipulate.

Parameters:ax (matplotlib axes) – The axes to be modified by this function.

Note

In the DefaultPlotStyle class this method is used to set SI units plotting mode for all axes.
