HDF5Folder¶

class Stoner.HDF5.HDF5Folder(*args, **kargs)[source]¶

Bases: object

A mixin class for Stoner.Folders.DataFolder that provides a method to load and save data from a single HDF5 file with groups.

See Stoner.Folders.DataFolder for documentation on constructor.

Datalayout consistns of sub-groups that are either instances of HDF5Files (i.e. have a type attribute that contains ‘HDF5File’) or are themsleves HDF5Folder instances (with a type attribute that reads ‘HDF5Folder’).
