BlochLaw¶

class Stoner.analysis.fitting.models.magnetism.BlochLaw(*args, **kwargs)[source]¶

Bases: lmfit.model.Model

Bloch’s law for spontaneous magnetism at low temperatures.

Parameters

  • T (array like) – Temperature (K)

  • g (float) – Lande g-factor

  • A (float) – The echange stiffness (Jm^{-1})

  • Ms (float) – Saturation moment (Am^{-1})

Returns

(array like) – Magnetisation values corresponding to the given temperatures.

Notes

Calculates :math:`1 - frac{left((Gamma(3/2) zeta(3/2)right)}

{(4pi^2)} (v_{ws} / S) (k_B T / D)^{3/2}`

This is the bulk version of Bloch’s law which is not fully correct for thin films. Model adapted from code by Dr Joseph Barker <j.barker@leeds.ac.uk>
