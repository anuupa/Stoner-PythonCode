Stoner.formats.facilities.BNLFile¶

class Stoner.formats.facilities.BNLFile(*args, **kargs)[source]¶

Reader of files in the SPEC format given by BNL (specifically u4b beamline but hopefully generalisable).

Author RCT 12/2011

The file from BNL must be split into seperate scan files before Stoner can use them, a separate python script has been written for this and should be found in data/Python/PythonCode/scripts.
