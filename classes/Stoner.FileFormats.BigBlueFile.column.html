BigBlueFile.column¶

BigBlueFile.column(col)¶

Extracts one or more columns of data from the datafile by name, partial name, regular expression or numeric index.

Parameters

col (int, string, list or re) – is the column index as defined for DataFile.find_col()

Returns

ndarray – One or more columns of data as a numpy.ndarray.
