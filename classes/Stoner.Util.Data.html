Stoner.Util.Data¶

class Stoner.Util.Data(*args, **kargs)[source]¶

A merged class of AnalyseFile and PlotFile which also has the FielFormats loaded redy for use. This ‘kitchen-sink’ class is intended as a convenience for writing scripts that carry out both plotting and analysis on data files.
