test_is_zip¶

Stoner.Zip.test_is_zip(filename, member='')[source]¶

Recursively searches for a zipfile in the tree.

Parameters:filename (str) – Path to test whether it is a zip file or not.
Keyword Arguments:
 member (str) – Used in recursive calls to identify the path within the zip file
Returns:False or (filename,member) – Returns False if not a zip file, otherwise the actual filename of the zip file and the nanme of the member within that zipfile.
