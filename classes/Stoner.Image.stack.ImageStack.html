Stoner.Image.stack.ImageStack¶

class Stoner.Image.stack.ImageStack(*args, **kargs)[source]¶

Stoner.Image.stack.ImageStack is a 3d numpy array stack of images.

This is used to deal with a stack of images with identical dimensions as a 3d numpy array - more efficient than the folder based methods in ImageFolder for iterating etc. The first axis is the number of images. Images are added and deleted through list type methods ImageStack.append, ImageStack.insert.

Images can be accessed through ImageStack[i] Metadata through ImageStack[‘key’] images should be added or removed using the list like methods append, insert, del
