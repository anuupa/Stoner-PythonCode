BNLFile.__init__¶

BNLFile.__init__(*params)[source]¶

Constructor modification.

Do a normal initiation using the parent class ‘self’ followed by adding an extra attribute line_numbers, line_numbers is a list of important line numbers in the file. I’ve left it open for someone to add options for more args if they wish.
