RasorFile.del_nan¶

RasorFile.del_nan(col=None, clone=False)¶

Remove rows that have nan in them.

eyword Arguments:
col (index types or None): column(s) to look for nan’s in. If None or not given, use setas columns. cone (boolean): if True clone the current object before running and then return the clone not self.
Returns:self (DataFile) – Returns a copy of the current object (or clone if *clone*=True)
