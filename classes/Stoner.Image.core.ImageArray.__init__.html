ImageArray.__init__¶

ImageArray.__init__(*args, **kwargs)[source]¶

Constructor method for ImageArray.

various forms are recognised

ImageArray('filename')

Creates the new ImageArray object and then loads data from the given filename.

ImageArray(array)

Creates a new ImageArray object and assigns the array.

ImageArray(ImageFile)

Creates a new ImageArray object and assigns the array using the ImageFile.image property.

ImageArray(ImageArray)

Creates the new ImageArray object and initialises all data from the existing ImageArray instance. This on the face of it does the same as the assignment operator, but is more useful when one or other of the ImageArray objects is an instance of a sub - class of ImageArray

ImageArray(bool)¶
:noindex:

if arg[0] is bool and False then open a file dialog to locate the file to open.

Parameters:arg (positional arguments) – an argument that matches one of the definitions above
Keyword Arguments: All keyword arguments that match public attributes are

used to set those public attributes eg metadata.

asfloat(bool):
if True and loading the image from file, convert the image to float values between 0 and 1 (necessary for some forms of processing)
