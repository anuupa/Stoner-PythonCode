DataFolder.sort¶

DataFolder.sort(key=None, reverse=False)[source]¶

Sort the files by some key

Keyword Arguments:
 key (string, callable or None) – Either a string or a callable function. If a string then this is interpreted as a metadata key, if callable then it is assumed that this is a a function of one paramater x that is a Stoner.Core.DataFile object and that returns a key value. If key is not specified (default), then a sort is performed on the filename

reverse (bool): Optionally sort in reverse order

Returns:A copy of the current DataFolder object
