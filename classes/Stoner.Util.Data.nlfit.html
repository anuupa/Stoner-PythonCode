Data.nlfit(ini_file, func)¶

Parameters:
  • ini_file (string) – path to ini file with model
  • func (string or callable) – Name of function to fit with (as seen in FittingFuncs.py module in Stoner) or function instance to fit with
Returns:

AnalyseFile instance, matplotlib.fig instance (or None if plotting disabled in the inifile)
