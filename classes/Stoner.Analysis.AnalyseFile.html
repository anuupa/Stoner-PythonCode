Stoner.Analysis.AnalyseFile¶

class Stoner.Analysis.AnalyseFile(*args, **kargs)[source]¶

Stoner.Analysis.AnalyseFile extends Stoner.Core.DataFile with numpy and scipy passthrough functions.

Note

There is no separate constructor for this class - it inherits from DataFile
