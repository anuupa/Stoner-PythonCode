Stoner.FileFormats.BNLFile¶

class Stoner.FileFormats.BNLFile(*params)[source]¶

Creates BNLFile a subclass of DataFile that caters for files in the SPEC format given by BNL (specifically u4b beamline but hopefully generalisable).

Author Rowan 12/2011

The file from BNL must be split into seperate scan files before Stoner can use them, a separate python script has been written for this and should be found in data/Python/PythonCode/scripts.

Methods

__init__(*params) Constructor modification BNLFile(‘filename’) Do a normal initiation using the parent class ‘self’ followed by adding an extra attribute line_numbers, line_numbers is a list of important line numbers in the file.
