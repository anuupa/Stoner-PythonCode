ImageArray.save¶

ImageArray.save(filename, **kargs)[source]¶

Saves the image into the file ‘filename’.

Metadata will be preserved in .png format. fmt can be ‘png’ or ‘npy’ or ‘both’ which will save the file in that format. metadata is lost in .npy format but data is converted to integer format for png so that definition can be lost and negative values are clipped.

Parameters:filename (string, bool or None) – Filename to save data as, if this is None then the current filename for the object is used If this is not set, then then a file dialog is used. If filename is False then a file dialog is forced.
