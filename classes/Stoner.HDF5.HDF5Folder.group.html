HDF5Folder.group¶

HDF5Folder.group(key)¶

Take the files and sort them into a series of separate DataFolder objects according to the value of the key

Parameters:key (string or callable or list) – Either a simple string or callable function or a list. If a string then it is interpreted as an item of metadata in each file. If a callable function then takes a single argument x which should be an instance of a DataFile and returns some vale. If key is a list then the grouping is done recursively for each element in key.
Returns:A copy of the current DataFolder object in which the groups attribute is a dictionary of DataFolder objects with sub lists of files

If ne of the grouping metadata keys does not exist in one file then no exception is raised - rather the fiiles will be returned into the group with key None. Metadata keys that are generated from the filename are supported.
