PlotFile.sort¶

PlotFile.sort(order=None, reverse=False)¶

Sorts the data by column name. Sorts in place and returns a copy of the sorted data object for chaining methods.

Keyword Arguments:
 
  • order (column index or list of indices or callable function) – Represent the sort order If order is a callable function then it should take a two tuple arguments and return +1,0,-1 depending on whether the first argument is bigger, equal or smaller.
  • reverse (boolean) – If true, the sorted array isreversed.
Returns:

A copy of the sorted object
