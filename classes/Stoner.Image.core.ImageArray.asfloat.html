ImageArray.asfloat¶

ImageArray.asfloat(normalise=False, clip_negative=False)[source]¶

Return the image converted to floating point type.

If currently an int type then floats will be automatically normalised. If currently a float type then will normalise if normalise. If currently an unsigned int type then image will be in range 0,1 :keyword normalise: normalise the image to -1,1 :kwtype normalise: bool :keyword clip_negative: clip negative intensity to 0 :kwtype clip_negative: bool
