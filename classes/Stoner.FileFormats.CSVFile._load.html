CSVFile._load¶

CSVFile._load(filename=None, header_line=0, data_line=1, data_delim=', ', header_delim=', ', **kargs)[source]¶

Generic deliminated file loader routine.

Parameters:

filename (string or bool) – File to load. If None then the existing filename is used, if False, then a file dialog will be used.
Keyword Arguments:
 
  • header_line (int) – The line in the file that contains the column headers. If None, then column headers are auotmatically generated.
  • data_line (int) – The line on which the data starts
  • data_delim (string) – Thge delimiter used for separating data values
  • header_delim (strong) – The delimiter used for separating header values
Returns:

A copy of the current object after loading the data.
