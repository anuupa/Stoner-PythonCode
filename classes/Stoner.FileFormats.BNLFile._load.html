BNLFile._load¶

BNLFile._load(filename, *args, **kargs)[source]¶

BNLFile.load(filename)

Parameters:filename (string or bool) – File to load. If None then the existing filename is used, if False, then a file dialog will be used.
Returns:A copy of the itself after loading the data.

Notes

Overwrites load method in DataFile class, no header positions and data positions are needed because of the hash title structure used in BNL files.

Normally its good to use _parse_plain_data method from DataFile class to load data but unfortunately Brookhaven data isn’t very plain so there’s a new method below.
