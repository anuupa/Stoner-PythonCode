TexFormatter¶

class Stoner.plot.formats.TexFormatter[source]¶

Bases: matplotlib.ticker.Formatter

An axis tick label formatter that emits Tex formula mode code.

Formating is set so that large numbers are registered as times 10^{power} rather than ysing E notation.
