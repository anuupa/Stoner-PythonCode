hsl2rgb¶

Stoner.plot.hsl2rgb(h, s, l)[source]¶

Converts from hsl colourspace to rgb colour space with numpy arrays for speed.

Parameters

  • h (array) – Hue value

  • s (array) – Saturation value

  • l (array) – Luminence value

Returns

2D array (Mx3) of unsigned 8bit integers
