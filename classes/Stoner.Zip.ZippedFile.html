ZippedFile¶

class Stoner.Zip.ZippedFile(*args, **kargs)[source]¶

Bases: Stoner.Core.DataFile

A sub class of DataFile that sores itself in a zip file.

If the first non-keyword arguement is not an zipfile:ZipFile then initialises with a blank parent constructor and then loads data, otherwise, calls parent constructor.
