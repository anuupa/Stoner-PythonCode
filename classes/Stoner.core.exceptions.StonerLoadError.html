StonerLoadError¶

exception Stoner.core.exceptions.StonerLoadError[source]¶

An exception thrown by the file loading routines in the Stoner Package.

This special exception is thrown when one of the subclasses of Stoner.Core.DataFile attmpts and then fails to load some data from disk. Generally speaking this is not a real error, but simply indicates that the file format is not recognised by that particular subclass, and thus another subclass should have a go instead.
