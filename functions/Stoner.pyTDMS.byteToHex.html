byteToHex¶

Stoner.pyTDMS.byteToHex(byteStr)[source]¶

Convert a byte string to it’s hex string representation e.g. for output.

Parameters:byteStr (string) – Input string of bytes

Note

Uses list comprehension which is a fractionally faster implementation than the alternative, more readable, implementation below:: hex = [] for aChar in byteStr:

hex.append( “%02X ” % ord( aChar ) )

return ‘’.join( hex ).strip()
